# DuckBot

## Prerequisites
- Python 3.8+
- Discord Developer Token

## How To Run

- instal packages
 
```bash
pip install discord requests python-decouple unicodedata
```

- Copy and renamed the env to .env
- Edit the .env file to your personal Discord bot token
- Run the bots
